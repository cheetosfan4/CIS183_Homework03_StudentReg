# CIS183_Homework03_StudentReg
## RGB Color Selector, Due 11/09/2025
This program allows the user to manage a database of students. The user is able to view and edit details of students, add new students, delete students, and search through the database of students using filters. They are also able to add new majors that they can use when adding or editing students. There are buttons on the main screen to add new students, new majors, and search through students. On the main screen and the search screen, the user can click on a student to view their details, and long click on a student to delete them.
